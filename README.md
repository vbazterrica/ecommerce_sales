# Ecommerce Sales Dashboard 📊

Interactive dashboard for analyzing ecommerce sales performance and business insights.

---

## 📌 Project Overview

This project is a data analytics dashboard built with Python and Streamlit.  
It allows users to explore ecommerce sales data and extract key business insights such as revenue trends, product performance, and customer behavior.

---

## 🚀 Features

- Sales and revenue analysis
- KPI tracking (orders, revenue, average order value)
- Product performance insights
- Time series analysis
- Interactive filters and visualizations
